## Description

security_tools is a repository created to serve as a personal laboratory. This is an open-source project focused on developing simplified security tools for learning and automation purposes.

The tools contained here are designed to be executed in controlled testing environments and for cybersecurity studies.

**WARNING:** Do not use these tools for illicit or illegal purposes. Ethical use and prior consent are fundamental in any security-related activity.
